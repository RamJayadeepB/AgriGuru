# Crop Recommendation Streamlit App

This repo contains a Streamlit app that recommends crops based on input features.
Files:
- `app.py` — Streamlit application.
- `model.joblib`, `encoder.joblib`, `model_columns.joblib` — trained artifacts (Option A).
- `crop_images/` — images displayed.
- `Dockerfile`, `start.sh`, `requirements.txt` — for containerized deployment.

Deploy instructions: push to GitHub and connect this repo to Render (Web Service, Docker). Streamlit runs on port 8501.
